# jsoo-svg-pan-zoom

js_of_ocaml interface to
[svg-pan-zoom](https://github.com/ariutta/svg-pan-zoom).

# Requires

1. opam install js_of_ocaml js_of_ocaml-ppx
2. [svg-pan-zoom 3.6.0](https://github.com/ariutta/svg-pan-zoom)
